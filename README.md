# 📝 To-Do List App

A beautifully styled, responsive, and theme-toggle-enabled To-Do List application built using **React.js**. 

This app allows users to:

- Add tasks
- Delete tasks
- Move tasks up or down
- Switch between light and dark themes


##  Tech Stack

- **React.js**
- **CSS3**
- **React Icons**
